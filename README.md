# Diagramme de Gantt — Correcteur de CV ATS

## Déploiement GitHub Pages

1. Téléverse `index.html` à la racine du dépôt
2. Va dans **Settings > Pages**
3. Choisis **Deploy from a branch**
4. Sélectionne **main** et **/(root)**
5. Enregistre

Le site sera accessible quelques instants après le déploiement.
