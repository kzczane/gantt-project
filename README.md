# Gantt Premium — Correcteur de CV ATS

## Déploiement GitHub Pages

1. Crée un dépôt GitHub
2. Ajoute le fichier `index.html` à la racine
3. Push sur `main`
4. Va dans **Settings > Pages**
5. Choisis **Deploy from a branch**
6. Sélectionne **main** et **/(root)**
7. Enregistre

Le site sera publié après quelques instants.
